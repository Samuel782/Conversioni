Conversione Binario Decimale

  • Che cosa è il sistema binario e il sistema decimale?

Il sistema binario è un sistema numerico a base 2, dove ogni cifra (bit) può assumere solo due valori: 0 o 1.

Il sistema decimale è un sistema numerico a base 10, il più comune, dove ogni cifra può assumere uno dei dieci valori da 0 a 9.

  • Come si converte un numero binario in decimale?

  1. Identifica le posizioni delle cifre binarie: Ogni cifra (bit) in un numero binario ha una posizione che parte da 0 per il bit più a destra.

    Per esempio, nel numero binario 1011, le posizioni sono:

    Posizione: 3 2 1 0
    Bit:               1 0 1 1

  2. Potenze di 2 corrispondenti:

    Potenza:     2^4   2^3   2^2   2^1   2^0
  3. Moltiplica ciascun bit per la potenza di 2 corrispondente:

    1 * 2^4 = 1 * 16 = 16
    1 * 2^3 = 1 * 8 = 8
    0 * 2^2 = 0 * 4 = 0
    1 * 2^1 = 1 * 2 = 2
    0 * 2^0 = 0 * 1 = 0

  4. Somma i risultati:

    16 + 8 + 0 + 2 + 0 = 26

Inserisci numero binario

  • Come si converte un numero decimale in binario?

  1. Dividi il numero decimale per 2: Prendi il numero decimale e dividilo per 2, annotando sia il quoziente che il resto.
  2. Annota il resto: Il resto sarà o 0 o 1 e farà parte del numero binario.
  3. Ripeti con il quoziente: Prendi il quoziente ottenuto e dividilo nuovamente per 2, annotando ancora il resto.
  4. Continua fino a ottenere un quoziente di 0: Continua questo processo fino a quando il quoziente diventa 0.
  5. Leggi i resti al contrario: Il numero binario sarà composto dai resti letti dall'ultimo al primo ottenuto.

Esempio Pratico:

Convertiamo il numero 13 in binario:

Dividendo Divisore Quoziente Resto
13 2 6 1
6 2 3 0
3 2 1 1
1 2 0 1

Ora leggiamo i resti al contrario: 1101
